Demo Name Model3D
Classes demonstrated

TModel3D
Description This demo demonstrates how to use TModel3D to load 3D models in FireMonkey applications at runtime.
Usage Load a 3D model from a file ( *.obj, *.dae, *.ase ) and change or view the material that the 3D model uses. For material you can use solid colors, textures or light materials.
Interaction You can interact with the loaded 3D model using mouse movement ( for rotation ) and scroll ( for zoom ).
Note Some models may not be visible immediately after being loaded due to the scale of the respective model. To view the model just use the track bar to change the scale. Also, some models may have a high level of detail so the loading operation may take longer; watch the status bar to see when the model has been loaded.
