Tethering demos
---------------

Photo Frame / DesktopWall
-------------------------
DesktopWall is a desktop application which receives photos from other devices.
The Mobile directory project can be run on a device with a camera to create 
photos and send them to the DesktopWall application. 

First run DesktopWall, then run MobilePhotoApp, create a photo by clicking on 
the photo icon and it will be sent to the DesktopWall application.
