TaskbarDemo project group has 3 projects and their corresponding CPP projects.

OneFormApp.dproj contains a form showing how to use the progress, overlay icons, buttons and clip region.

MDIApp.dproj is an MDI application where each new MDI child form is shown as a new tab preview on the 
task bar, showing the content of all MDI open forms.

VideoPlayer.dproj is a very simple media player, using two buttons on the taskbar to invoke play/pause actions 
for the opened video.
