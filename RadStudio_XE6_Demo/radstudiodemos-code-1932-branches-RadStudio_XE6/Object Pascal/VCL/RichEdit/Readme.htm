Description The demo demonstrates the main features of TRichEdit class.

A rich edit control is a window in which the user can enter and edit text. The text can be assigned character and paragraph formatting, and can include embedded OLE objects. Rich edit controls provides a programming interface for formatting text (see ..\overview.rtf for additional details.)
