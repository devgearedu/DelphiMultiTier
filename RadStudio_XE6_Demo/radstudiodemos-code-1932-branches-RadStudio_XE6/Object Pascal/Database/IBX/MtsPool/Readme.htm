Description This project demonstrates the use of object pooling.
The application measures the amount of time needed to open and close a database connection a number of times. The "MTS Pooling checkbox" enables and disables the database objects pooling. You will see a difference in elapsed time when running the test with and without MTS Pooling enabled.


This example requires that Interbase XE be installed. Make sure the server is running before you run the example.
