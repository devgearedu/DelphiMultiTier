BorlandIDEServices

Open Tools API (OTA)
Description This demo shows how to extend Delphi's IDE editor including graphics control, tab insertion, and options settings. After loading TestEditorViewAPI.dproj, install the package by right clicking on TestEditorViewAPI.bpl in the project manager and picking Install. Once the build and install is complete, the package will add the following:
  • A new IDE menu item, View | Bouncy Ball, which will create a new editor tab with a bouncing ball, and an associated Close BouncyBall item.
  • A new tab called "OS File Information" holding file data related to the units represented by currently shown edit tabs.
