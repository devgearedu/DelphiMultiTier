and other interfaces of ToolsAPI library.
Description This is an IDE keyboard binding expert that provides the following functions.
  • C++, Delphi, XML parser.
  • Region Expert - marks a block of selected text as a region with a keystroke in Delphi.
  • Comment Expert - marks a block of selected text as comments with a keystroke in Delphi.
  • generates XML-styled comments (XML Doc) for code declarations with a key sequence in Delphi and C#.
  • auto comments extension expert - automatically inserts // in between blocks of lines in Delphi.
  • Goto Region Expert - iterates through all open files and presents a list of regions, allowing you to select what region to go to, optionally expanding the region, and displaying the selected region at the top of the editor. Activate using Ctrl O + Ctrl X ( NOT Ctrl O + X ).<
  • MSDN Help expert - provides help using Ctrl F1 by looking up in MSDN with the word nearest the cursor.
