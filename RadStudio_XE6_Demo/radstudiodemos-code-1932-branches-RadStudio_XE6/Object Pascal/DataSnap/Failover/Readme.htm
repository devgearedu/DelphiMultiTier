Description Steps to run:
  • Open FailoverDemo.groupproj
  • Compile and run Failover project
  • Compile and run MyServer project, define the port 211 and start
  • Compile and run MyServer project, define the port 213 and start

    To run DataExplorer as the client:

  • 1 - Create a DataSnap connection using DataExplorer
  • 2 - Modify the DataSnap connection created on DataExplorer to Protocol = HTTP Server = localhost Port = 8020 User = admin Password = admin
  • 3 - Search for the Server Method (Procedure on DataExplorer) TServerMethod1.EchoString
  • 4 - Define the Value (i.e. Delphi) click the execute button - it will return Delphi 211
  • 5 - Shutdown the server connected to port 211
  • 6 - Re-execute step 4 - it will return Delphi 213
