Demo Name Multi Language Demo
Classes demonstrated

TLang

Description This demo shows how to create a multilanguage application. It was created by adding a TLang component onto a form, then double clicking it to invoke the Language Designer, which will scan the strings used and allow additional translations, languages, and templates to saved.

All components have a Boolean property “auto translate” that causes strings to be translated automatically If TLang.AutoSelect=true, FireMonkey selects the language using the current system locale.
