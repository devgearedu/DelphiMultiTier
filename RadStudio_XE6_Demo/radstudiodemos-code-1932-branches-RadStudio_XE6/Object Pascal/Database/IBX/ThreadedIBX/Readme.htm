Demo Name ThreadedIBX
Classes demonstrated

TIBDatabase

TIBTransaction

TIBQuery

TDataSource

TDBGrid

TStringGrid

Description This demo shows possibility of executing and displaying results of query to a database in a parallel mode. Results are output to the DBGrid and the TStringGrid. The user can choose where to display results of query execution.

This example requires that Interbase XE be installed. Make sure the server is running before you run the example.
